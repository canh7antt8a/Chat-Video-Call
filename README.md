## Cài đặt
Yêu cầu: cài NodeJs
``` 
# Dùng terminal di chuyển đến thư mục source code

# Cài modules
npm install

# Server: localhost:3000
npm app.js
```
Truy cập `` http://localhost:3000`` Để vào ứng dụng.
